# namp2
Not another MP2 a.k.a MP3 done the right way.

# Requirements.
- PHP 5.3.6

# OSX Install.
- Install MAMP.
- Install Sequel Pro.
- Install WordPress.
- Enable debug mode in wp-settings.php
```php
define(WP_DEBUG, true);
```
- ...

# Rules.
- Two spaces indentation, Ruby style.
- CamelCase for classes.
- snake_case for methods.
- ...
