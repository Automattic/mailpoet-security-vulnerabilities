# namp2

Not another MP2 a.k.a MP3 done the right way.

# Requirements.

- PHP 5.3.6

# Install.

- Clone the repo in wp-content/plugins.

# Structure.

- /assets
CSS and JS.

- /deps
PHP dependencies.

- /lang
Language files.

- /lib
MailPoet classes.

- /test
Acceptance and spec tests.

- /mailpoet.php
Kickstart file.

# Rules.

- Two spaces indentation, Ruby style.
- CamelCase for classes.
- snake_case for methods.
- ...
